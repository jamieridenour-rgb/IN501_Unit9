# IN501_Unit9
Group Project Unit 9

Assignment Instructions

Meet with your group.
For this assignment, you will work in small groups as assigned by your instructor.
Decide which group member will serve as team lead. This member will submit the zip file containing all portions of the project.
Create a Baseline CNN Model
Load and preprocess your MNIST dataset.
Use a tool, such as Matplotlib, to display the first 10 digit images from the dataset.
Create a Sequential model adding necessary layers.
Train the model on the training data.
Evaluate the model on the test data and print out the loss and accuracy.
Use your model to generate predictions based on your test data.          
Repeat step b in this list but this time include the corresponding prediction from your model as well.
Take one or more screenshots of your output window and any dialog boxes generated. Be sure to show the test data’s loss and accuracy as well as the results of evaluating the 10 images.
Develop Individual Variations
Each group member must modify at least one “setup parameter”  in the CNN.
Explain what the parameter does and provide a brief hypothesis about how you expect your change to affect the model (e.g., “increasing filters should improve accuracy, but might take longer to train”).
Each group member trains, evaluates, and documents their modified model as was done with the baseline program built in step 2.
Group Discussion and Comparison
After each member completes their variation, hold a team discussion where everyone shares:
What they changed.
Their hypothesis.
Their actual results.
Comparison to other group members’ changes and baseline model.
Prepare a Group Presentation
Create a PowerPoint presentation which includes a slide for each of the following:
Title slide including the names of each member of the group.
Baseline Model: Description of its setup and results.
At least one slide for each variation to include:
Name of group member for this variation
Parameters changed in this variation
Hypothesis to include expected effect on performance, runtime, etc.
Test Results and how they compare with both the hypothesis and the baseline results.
A summary slide comparing the results from all versions of the program.
Peer Review Form
Each team member (including the selected team lead) must submit a completed Peer Review Form.

Directions for Submitting Your Assignment

Team leader will submit the program source code file (.py) for each version of the program, your screenshot images, and your presentation file in a single zip file.  The .zip file should be named IN501_Each_Member_Last_Name_Unit9_Assignment.

All members will submit the completed Peer Review Form. The name format of this document will be IN501_YourName_Unit9_PeerReview.docx
